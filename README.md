# 青空しおり

青空文庫の著作権が切れた作品を、N2〜N1の日本語学習者が「読み切れる」形に整えるオープンソースの読書サイトです。

## 現在できること

- N2中心・N1までの10作品を、レベル・長さ・ジャンルで選ぶ
- ルビの表示切替、学習片と原文全文の切替
- 文中の語をタップして意味を確認し、復習へ追加
- 読書進捗・語彙・簡易統計を端末内に保存
- 青空文庫の原文・出典を各作品に表示
- PC / スマートフォン対応

Googleログインと端末間同期は、OAuthクライアントとCloudflare D1を接続する次の段階です。現在の公開版は登録不要で利用でき、データはブラウザ内だけに保存されます。

## 開発

```bash
npm install
npm run import:aozora
npm run dev
```

`import:aozora` は既定で `/Volumes/minipc-1/git/aozorabunko` を読みます。別の場所なら `AOZORA_ROOT` を指定してください。リポジトリ全体を走査せず、選定済みのファイルだけを読み込みます。

## 検証とデプロイ

```bash
npm test
npm run lint
npm run build
npm run deploy
```

公開版は Cloudflare Pages の静的配信を利用しています。Workers 用の SPA 設定も残しているため、将来ログイン同期 API を追加した時に移行できます。初期版は動的 API を使わないため、通常の個人利用なら無料枠で運用できます。
